# Contributing to VDisplay

## License reminder

- Personal / non-commercial use is free.
- Commercial use requires a paid license ([LICENSE](LICENSE)).
- By opening a PR you agree your contribution is under the same license.

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/my-change`
3. Build: `dotnet build VDisplay.sln -c Release` (and driver if you touch IDD)
4. Keep PRs focused; update `docs/` if behavior changes
5. Open a Pull Request against `main`
